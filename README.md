# malloy-service

A simple GRPC based server for the [Malloy](https://github.com/malloydata/malloy/) language.
Currently used by [malloy-py](https://github.com/malloydata/malloy-py).

## Building

### Development

```
npm install
npm run build
```

or

```
npm install
npm run start
```

### Packaging

```
npm run package
```
